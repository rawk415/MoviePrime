MoviePrime
=====
영화 커뮤니티와 영화 정보를 결합한 간단한 웹사이트

### 맡은 역할
- 주제 선정
- DB 설계 및 연동
- 웹 크롤링(네이버 영화)
- 게시판, 영화 정보 구현
- 회원가입, 로그인, 이메일 인증 구현

### 소프트웨어 정보
`tomcat 9.0.30` `jdk 1.8` `MySQL`
